Why do we need Ext.onReady?

  • JavaScript should only run when the DOM is loaded
  • Browser ready state events are inconsistent
    • FireFox and Opera fire DOMContentLoaded event
    • IE Requires a script tag in the body with a defer attribute
    • Safari fires no event, sets document.readyState to 'complete'
  • Ext Levels the playing field!
  • Fires your code at just the right time
