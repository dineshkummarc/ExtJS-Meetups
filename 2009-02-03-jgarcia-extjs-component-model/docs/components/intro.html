Ext's Component Model

  • Provides a central model for UI components
  • Full lifecycle
    • Initialization
    • Render
    • Destruction
  • Utility Methods, such as:
    • findParentBy
    • getXType
    • focus
  • Central event model that allows for prevention of code execution
    • beforerender/render
    • beforedestroy/destroy
    • beforehide/hide
    • beforeshow/show
  • Lazy Instantiation with XTypes
